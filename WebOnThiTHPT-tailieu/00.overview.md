# <a name="tổng-quan-hệ-thống-web-ôn-thi-thpt"></a>00 — Tổng quan hệ thống Web Ôn Thi THPT
Web Ôn Thi THPT là nền tảng học tập trực tuyến cho phép học sinh luyện đề, làm bài thi thử, theo dõi kết quả học tập và hỗ trợ giáo viên quản lý ngân hàng câu hỏi, đề thi và kết quả học sinh.

-----
## <a name="mục-tiêu"></a>Mục tiêu

|Mục tiêu|Mô tả|
| :- | :- |
|Luyện đề trực tuyến|Học sinh có thể làm bài thi mọi lúc, mọi nơi|
|Mô phỏng kỳ thi thật|Trải nghiệm thi tương tự kỳ thi THPT Quốc Gia|
|Quản lý ngân hàng câu hỏi|Giáo viên quản lý tập trung câu hỏi và đề thi|
|Chấm điểm tự động|Hệ thống tự động tính điểm và lưu kết quả|
|Theo dõi tiến độ học tập|Học sinh xem lịch sử làm bài và kết quả|
|API-first|Backend cung cấp API cho Web, Mobile và các hệ thống khác|

-----
## <a name="scope-mvp"></a>Scope MVP
### <a name="trong-scope"></a>Trong scope
#### <a name="authentication"></a>*Authentication*
- Đăng ký tài khoản
- Đăng nhập
- JWT Authentication
- Refresh Token
- Quên mật khẩu
#### <a name="user-management"></a>*User Management*
- Quản lý hồ sơ người dùng
- Phân quyền Student / Teacher / Admin
#### <a name="question-management"></a>*Question Management*
- CRUD câu hỏi
- Phân loại theo môn học
- Phân loại theo chương
- Phân loại theo độ khó
#### <a name="exam-management"></a>*Exam Management*
- Tạo đề thi
- Chỉnh sửa đề thi
- Xóa đề thi
- Gán câu hỏi vào đề thi
#### <a name="exam-taking"></a>*Exam Taking*
- Học sinh làm bài thi
- Giới hạn thời gian làm bài
- Nộp bài thi
#### <a name="result-management"></a>*Result Management*
- Chấm điểm tự động
- Xem kết quả
- Xem lịch sử làm bài
- Xem đáp án
#### <a name="dashboard"></a>*Dashboard*
- Thống kê số lượng học sinh
- Thống kê số lượng đề thi
- Thống kê kết quả học tập
