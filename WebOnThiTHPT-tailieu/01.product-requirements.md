# <a name="product-requirements"></a>01 — Product Requirements
Tài liệu mô tả các yêu cầu nghiệp vụ và chức năng cấp cao của hệ thống Web Ôn Thi THPT.

-----
# <a name="mục-tiêu-sản-phẩm"></a>1. Mục tiêu sản phẩm
Hệ thống Web Ôn Thi THPT được xây dựng nhằm hỗ trợ học sinh luyện tập và đánh giá năng lực trước kỳ thi THPT Quốc Gia thông qua việc làm đề thi trực tuyến, đồng thời hỗ trợ giáo viên quản lý ngân hàng câu hỏi và đề thi.

-----
# <a name="bài-toán-cần-giải-quyết"></a>2. Bài toán cần giải quyết
## <a name="đối-với-học-sinh"></a>Đối với học sinh
### <a name="hiện-trạng"></a>Hiện trạng
- Khó tìm nguồn đề thi chất lượng
- Không có môi trường mô phỏng kỳ thi thật
- Khó theo dõi tiến độ học tập
- Không biết điểm mạnh và điểm yếu của bản thân
### <a name="mong-muốn"></a>Mong muốn
- Làm đề thi trực tuyến
- Chấm điểm tự động
- Theo dõi lịch sử học tập
- Xem đáp án và lời giải
-----
## <a name="đối-với-giáo-viên"></a>Đối với giáo viên
### <a name="hiện-trạng-1"></a>Hiện trạng
- Quản lý câu hỏi rời rạc
- Khó tạo đề thi nhanh
- Khó theo dõi kết quả học sinh
### <a name="mong-muốn-1"></a>Mong muốn
- Quản lý tập trung ngân hàng câu hỏi
- Tạo đề thi dễ dàng
- Theo dõi kết quả học sinh
-----
## <a name="đối-với-quản-trị-viên"></a>Đối với quản trị viên
### <a name="hiện-trạng-2"></a>Hiện trạng
- Khó quản lý dữ liệu người dùng
- Thiếu thống kê hệ thống
### <a name="mong-muốn-2"></a>Mong muốn
- Quản lý người dùng
- Quản lý môn học
- Theo dõi hoạt động hệ thống
-----
# <a name="phạm-vi-sản-phẩm-mvp"></a>3. Phạm vi sản phẩm (MVP)
## <a name="bao-gồm"></a>Bao gồm
### <a name="authentication"></a>Authentication
- Đăng ký tài khoản
- Đăng nhập
- Đăng xuất
- Quên mật khẩu
- Đổi mật khẩu
### <a name="user-management"></a>User Management
- Quản lý hồ sơ cá nhân
- Cập nhật thông tin cá nhân
### <a name="question-management"></a>Question Management
- Tạo câu hỏi
- Chỉnh sửa câu hỏi
- Xóa câu hỏi
- Tìm kiếm câu hỏi
### <a name="exam-management"></a>Exam Management
- Tạo đề thi
- Chỉnh sửa đề thi
- Xóa đề thi
- Công khai hoặc ẩn đề thi
### <a name="exam-taking"></a>Exam Taking
- Bắt đầu bài thi
- Trả lời câu hỏi
- Nộp bài thi
- Tự động hết giờ
### <a name="result-management"></a>Result Management
- Chấm điểm tự động
- Lưu lịch sử làm bài
- Xem kết quả chi tiết
### <a name="dashboard"></a>Dashboard
- Thống kê người dùng
- Thống kê đề thi
- Thống kê kết quả
-----
## <a name="không-bao-gồm"></a>Không bao gồm
- AI sinh đề thi
- AI trợ giảng
- Thanh toán
- Mobile App
- Livestream học tập
- Thi đấu trực tuyến
-----
# <a name="vai-trò-người-dùng"></a>4. Vai trò người dùng
## <a name="student"></a>Student
Mục tiêu:

- Luyện đề
- Theo dõi kết quả

Khả năng:

- Làm bài thi
- Xem kết quả
- Xem lịch sử làm bài
- Quản lý hồ sơ cá nhân
-----
## <a name="teacher"></a>Teacher
Mục tiêu:

- Quản lý nội dung học tập

Khả năng:

- Quản lý câu hỏi
- Quản lý đề thi
- Theo dõi kết quả học sinh
-----
## <a name="admin"></a>Admin
Mục tiêu:

- Quản trị hệ thống

Khả năng:

- Quản lý người dùng
- Quản lý danh mục
- Xem thống kê hệ thống
-----
