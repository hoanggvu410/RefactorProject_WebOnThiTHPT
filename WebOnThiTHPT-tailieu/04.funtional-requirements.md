# <a name="auth-authentication"></a>§AUTH — Authentication
## <a name="auth-01-đăng-ký"></a>AUTH-01: Đăng ký
- POST /v1/auth/register nhận email, password, full\_name
- Validate email format, password tối thiểu 8 ký tự
- Kiểm tra email chưa tồn tại trong hệ thống
- Hash password bằng bcrypt (rounds=12)
- Tạo user với:
  - email\_verified=false
  - is\_active=true
- Sinh OTP 6 chữ số
- Lưu OTP vào Redis:
  - key: otp:{email}
  - TTL: 15 phút
- Gửi OTP qua email
- Return user\_id + message hướng dẫn xác thực email
-----
## <a name="auth-02-xác-thực-email"></a>AUTH-02: Xác thực email
- POST /v1/auth/verify-email nhận email, otp
- Kiểm tra OTP trong Redis:
  - key: otp:{email}
- Nếu OTP đúng:
  - set email\_verified=true
  - xóa OTP khỏi Redis
  - phát access token + refresh token
- Nếu OTP sai:
  - trả lỗi AUTH\_INVALID\_OTP
- Nếu OTP hết hạn:
  - trả lỗi AUTH\_OTP\_EXPIRED
-----
## <a name="auth-03-đăng-nhập"></a>AUTH-03: Đăng nhập
- POST /v1/auth/login nhận email, password, remember\_me
- Check Redis key:
  - account\_locked:{email}
- Nếu key tồn tại:
  - trả lỗi AUTH\_ACCOUNT\_LOCKED
- Verify bcrypt password
- Nếu password sai:
  - INCR login\_attempts:{email}
  - Window: 10 phút
  - Sau 5 lần sai:
    - khóa tài khoản 15 phút
    - tạo key account\_locked:{email}
- Kiểm tra:
  - email\_verified=true
  - is\_active=true
- Phát access token JWT
- Phát refresh token
- Access token TTL: 15 phút
- Refresh token TTL:
  - 7 ngày (mặc định)
  - 30 ngày (remember\_me=true)
-----
## <a name="auth-04-refresh-token"></a>AUTH-04: Refresh Token
- POST /v1/auth/refresh nhận refresh\_token
- Lookup SHA256(refresh\_token) trong database
- Kiểm tra:
  - revoked\_at IS NULL
  - expires\_at > NOW()
- Revoke refresh token cũ ngay lập tức (RTR)
- Phát access token mới
- Phát refresh token mới
- Lưu refresh token mới vào database
-----
## <a name="auth-05-đăng-xuất"></a>AUTH-05: Đăng xuất
- POST /v1/auth/logout
- Yêu cầu access token hợp lệ
- Revoke refresh token hiện tại
- Xóa session hiện tại
-----
## <a name="auth-06-đăng-xuất-tất-cả-thiết-bị"></a>AUTH-06: Đăng xuất tất cả thiết bị
- POST /v1/auth/logout-all
- Revoke toàn bộ refresh token của user
- Buộc đăng nhập lại trên tất cả thiết bị
-----
## <a name="auth-07-quên-mật-khẩu"></a>AUTH-07: Quên mật khẩu
- POST /v1/auth/forgot-password nhận email
- Kiểm tra email tồn tại
- Sinh OTP reset password 6 chữ số
- Lưu Redis:
  - key: reset\_password:{email}
  - TTL: 15 phút
- Gửi OTP qua email
-----
## <a name="auth-08-đặt-lại-mật-khẩu"></a>AUTH-08: Đặt lại mật khẩu
- POST /v1/auth/reset-password nhận email, otp, new\_password
- Kiểm tra OTP trong Redis
- Validate password tối thiểu 8 ký tự
- Hash password bằng bcrypt (rounds=12)
- Cập nhật mật khẩu mới
- Xóa OTP khỏi Redis
- Revoke toàn bộ refresh token hiện có
-----
