# Environment Setup

Tài liệu này hướng dẫn thiết lập môi trường chạy local cho dự án **Web Ôn Thi THPT**.

## 1. Yêu cầu hệ thống

- Python 3.10+ 
- PostgreSQL
- `pip` và `venv`

## 2. Clone source code

```bash
git clone <repo-url>
cd RefactorProject_WebOnThiTHPT
```

## 3. Tạo virtual environment

### Linux / Ubuntu / macOS

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

## 4. Cài dependencies

```bash
pip install -r requirement.txt
```

## 5. Cấu hình biến môi trường

Tạo file `.env` ở thư mục gốc dự án, có thể copy từ `.env.example`:

```env
DATABASE_URL=postgresql://postgres:123456@localhost:5432/postgres
SECRET_KEY=your_secret_key
REFRESH_SECRET_KEY=your_refresh_secret_key
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=15
REFRESH_TOKEN_EXPIRE_DAYS=30
```

Ghi chú:
- `DATABASE_URL` được dùng ở `app/base/db.py`
- `SECRET_KEY` và `REFRESH_SECRET_KEY` được dùng cho JWT
- `ALGORITHM` mặc định là `HS256`
- `ACCESS_TOKEN_EXPIRE_MINUTES` mặc định là `15`
- `REFRESH_TOKEN_EXPIRE_DAYS` mặc định là `30`

## 6. Tạo database PostgreSQL

Tạo database tương ứng với `DATABASE_URL`, ví dụ:

```sql
CREATE DATABASE postgres;
```

Nếu bạn đổi tên database trong `.env`, hãy tạo đúng database đó trước khi chạy ứng dụng.

## 7. Chạy ứng dụng

```bash
uvicorn main:app --reload
```

Ứng dụng sẽ chạy tại:

- `http://127.0.0.1:8000`
- Swagger UI: `http://127.0.0.1:8000/docs`

## 8. Ghi chú về frontend

Frontend tĩnh được phục vụ trực tiếp từ backend:

- Trang chính: `/`
- Static files: `/static`

## 9. Lỗi thường gặp

- **Không kết nối được PostgreSQL**: kiểm tra `DATABASE_URL`, user/password và port.
- **Thiếu SECRET_KEY**: đảm bảo file `.env` đã có `SECRET_KEY` và `REFRESH_SECRET_KEY`.
- **Bảng chưa tồn tại**: ứng dụng hiện gọi `Base.metadata.create_all(bind=engine)` khi khởi động, nên database phải truy cập được trước khi chạy server.
