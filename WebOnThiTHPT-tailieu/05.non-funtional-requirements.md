# <a name="non-functional-requirements"></a>05 — Non-Functional Requirements
-----
# <a name="performance-performance"></a>§PERFORMANCE — Performance
- 95% API response dưới 500ms.
- 99% API response dưới 1 giây.
- Hệ thống hoàn thành chấm điểm và trả kết quả trong vòng 5 giây sau khi học sinh nộp bài.
- Danh sách đề thi và kết quả thi phải tải dưới 2 giây.
-----
# <a name="availability-availability"></a>§AVAILABILITY — Availability
- Hệ thống đạt uptime tối thiểu 99%.
- Dữ liệu được sao lưu tự động mỗi ngày.
- Dịch vụ có khả năng tự khởi động lại khi gặp sự cố.
- Không làm mất dữ liệu bài thi đã nộp.
-----
# <a name="security-security"></a>§SECURITY — Security
- Mật khẩu được mã hóa bằng bcrypt trước khi lưu vào cơ sở dữ liệu.
- Hệ thống sử dụng JWT Authentication cho tất cả API được bảo vệ.
- Access Token có thời hạn 15 phút, Refresh Token có thời hạn 7 ngày.
- Toàn bộ môi trường Production bắt buộc sử dụng HTTPS.
- Người dùng phải xác thực email trước khi sử dụng hệ thống.
- Giới hạn số lần đăng nhập sai để chống brute-force attack.
- Không lưu mật khẩu hoặc dữ liệu nhạy cảm dưới dạng plain text.
- Tất cả endpoint phải kiểm tra quyền truy cập theo vai trò người dùng.
-----
# <a name="scalability-scalability"></a>§SCALABILITY — Scalability
- Hệ thống hỗ trợ tối thiểu 500 người dùng đồng thời.
- Backend hỗ trợ triển khai nhiều instance.
- Redis được sử dụng cho OTP, Refresh Token, Rate Limiting và Cache.
- Thiết kế cơ sở dữ liệu hỗ trợ mở rộng số lượng người dùng, đề thi và kết quả thi trong tương lai.
-----
# <a name="maintainability-maintainability"></a>§MAINTAINABILITY — Maintainability
- Backend áp dụng Layered Architecture.
- Tuân thủ chuẩn PEP8 và sử dụng Type Hints.
- API Documentation được sinh tự động bằng OpenAPI/Swagger.
- Toàn bộ lỗi hệ thống phải được ghi log.
- Unit Test Coverage tối thiểu 70%.
- Secret và cấu hình môi trường phải được quản lý thông qua biến môi trường.
-----
# <a name="compatibility-compatibility"></a>§COMPATIBILITY — Compatibility
- Hỗ trợ Google Chrome, Microsoft Edge và Firefox.
- Giao diện hoạt động trên Desktop, Tablet và Mobile.
- API sử dụng chuẩn RESTful và JSON.
- Hệ thống sử dụng PostgreSQL làm cơ sở dữ liệu chính.
- Hỗ trợ triển khai trên Linux và Docker.
-----
